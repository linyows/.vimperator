# Readme

## How to use

    $ cd ~/
    $ git clone git@github.com:linyows/.vimperator.git .vimperator
    $ ln -s .vimperator/.vimperatorrc .vimperatorrc
    or
    $ cp .vimperator/.vimperatorrc .vimperatorrc
